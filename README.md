# Netværksdiagnoseværktøj ![PowerShell](https://img.shields.io/badge/PowerShell-Script-blue)

Et PowerShell-script, der automatisk tjekker og diagnosticerer netværksforbindelser på din Windows-maskine.

Scriptet logger alt til en fil på skrivebordet og giver visuel feedback i konsollen, så du hurtigt kan se, om alt fungerer som det skal.

## Formål med projektet

At anvende PowerShell til noget praktisk og relevant for min datatekniker-uddannelse.

Projektet samler teori fra netværk og OSI-modellen med mit ønske om, at lave et værktøj til automatisk fejlfinding af netværksproblemer kodet i PowerShell.

## Funktioner

- Viser aktive netværksadaptere ( med navn, beskrivelse og linkhastighed)  
- Henter IP-konfiguration og default gateway  
- Pinger gateway og internet (8.8.8.8)  
- Tester DNS-servere og DNS-opslag  
- Tester HTTP/HTTPS  
- Traceroute ved netværksfejl  
- Logger automatisk til skrivebordet med alle resultater
- Fejlbeskeder med forslag til den mest sandsynlige løsning på eventuelle problemer
